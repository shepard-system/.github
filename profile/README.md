# shepard-system

**Self-hosted observability and governance for AI coding agents.**

One human. Multiple AI agents. Full visibility.

Named after [Commander Shepard](https://digitalshepard.ai/about/) — the human-in-the-loop commanding a squad of
specialists.

---

## Repositories

| Repo                                                                         | Alias       | What it does                                              |
|------------------------------------------------------------------------------|-------------|-----------------------------------------------------------|
| [**shepard-obs-stack**](https://github.com/shepard-system/shepard-obs-stack) | The Eye     | Observability stack + CLI hooks. Clone, configure, see.   |
| **shepard-core**                                                             | The Staff   | Orchestration, routing, execution                         |
| **shepard-knwl-lake**                                                        | The Pasture | Knowledge lake — rules, skills, context. Source of truth. |
| **shepard-mcp-hub**                                                          | The Gate    | Unified tool access. The only way out.                    |
| **shepard-mem-layer**                                                        | The Memory  | Consolidation + search. How the herd remembers.           |
